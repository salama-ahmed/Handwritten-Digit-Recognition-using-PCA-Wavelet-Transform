# Handwritten Digit Recognition using PCA & Wavelet Transform

A Machine Learning project for handwritten digit recognition using Principal Component Analysis (PCA) and Wavelet Transform for feature extraction and dimensionality reduction.

---

## Features

* Handwritten Digit Recognition
* Feature Extraction using Wavelet Transform
* Dimensionality Reduction using PCA
* Machine Learning Classification
* Data Preprocessing and Visualization
* Performance Evaluation

---

## Technologies

* Python
* NumPy
* Scikit-learn
* OpenCV
* Jupyter Notebook
* PCA
* Wavelet Transform

---

## Project Workflow

1. Load handwritten digit dataset.
2. Apply image preprocessing techniques.
3. Extract features using Wavelet Transform.
4. Reduce dimensionality using PCA.
5. Train Machine Learning models.
6. Evaluate classification performance.
7. Predict handwritten digits from test samples.

---

## Applications

* Optical Character Recognition (OCR)
* Document Digitization
* Automated Form Processing
* Educational AI Systems
* Pattern Recognition Applications
